# OpenClaw Control 使用文档

## 应用简介

OpenClaw Control 是一款专为 OpenClaw 服务设计的安卓客户端应用。它通过 WebView 技术，将 OpenClaw 服务以原生应用的形式呈现，提供更便捷的访问体验。

### 主要功能
- **智能连接验证**：自动检测并验证 OpenClaw 服务地址
- **安全登录管理**：本地保存登录凭证，支持自动登录
- **WebView 优化**：提供最佳化的原生态浏览体验
- **版本自动更新**：自动检测并提示应用更新

---

## 安装要求

### 系统要求
- **Android 版本**：Android 5.0 (API 21) 及以上
- **存储权限**：需要读写存储权限以保存配置信息
- **网络连接**：需要稳定的网络连接

### 所需权限
- `INTERNET` - 网络访问
- `READ_EXTERNAL_STORAGE` - 读取配置
- `WRITE_EXTERNAL_STORAGE` - 保存登录信息

---

## 快速开始
下载地址：[releases](https://github.com/eraycc/openclaw-android/releases)

### 1. 首次启动
安装完成后，点击应用图标启动。首次启动时会自动检查更新。

### 2. 登录设置
在登录页面填写以下信息：
- **OpenClaw 地址**：例如 `https://your-openclaw-server.com` 或 `http://192.168.1.100`
- **访问令牌**：您的 OpenClaw 访问密钥

### 3. 自动验证
应用会自动验证：
- 地址格式是否正确
- 是否为有效的 OpenClaw 服务
- 网络连接是否正常

### 4. 成功登录
验证通过后，登录信息会被安全保存，下次启动时会自动登录。

## 详细功能说明

### 更新检查机制
- 启动时自动检查 GitHub 上的版本信息
- 版本号对比（支持语义化版本）
- 显示更新日志、文件大小和 MD5 校验值
- 支持强制更新模式

### WebView 优化特性
为提供最佳浏览体验，应用对 WebView 进行了全面优化：

| 功能类别 | 具体设置 |
|---------|---------|
| JavaScript | 完全启用 |
| 存储支持 | DOM 存储、数据库存储 |
| 视口适配 | 宽度自适应、全局视图 |
| 缩放控制 | 支持缩放、隐藏缩放控件 |
| 文件访问 | 允许文件 URL 访问 |
| 混合内容 | 允许 HTTP/HTTPS 混合 |
| 性能优化 | 硬件加速、缓存策略 |
| 用户代理 | 添加 Chrome 标识 |

### 安全退出机制
为防止误触返回键，应用采用"三秒内连续按三次返回键"的退出机制：
- **第1次**：开始计时（3秒）
- **第2次**：提示即将退出
- **第3次**：确认退出应用

## 配置文件说明

### 配置文件位置
```
应用数据目录/config/login.json
```

### 配置文件内容
```json
{
  "openclaw_address": "您的OpenClaw服务器地址",
  "openclaw_pwd": "您的访问令牌"
}
```

### 配置保存机制
- 登录信息保存在应用私有目录
- 其他应用无法访问
- 卸载应用时会自动清除

## 更新机制详解

### 版本检查流程
1. 启动时检测网络连接
2. 从 GitHub 获取最新版本信息
3. 对比当前版本号
4. 根据比对结果执行相应操作

### 版本信息文件格式
```json
{
  "AppName": "应用名称",
  "PackageName": "包名",
  "AnnounceMent": "公告信息",
  "ReleaseNotes": "更新日志",
  "VersionCode": "版本号",
  "ReleaseVersion": "发布版本",
  "BetaVersion": "测试版本",
  "MinSupportedOS": "最低系统版本",
  "TargetSDK": "目标SDK",
  "DownloadUrl": "下载地址",
  "FileSize": "文件大小",
  "MD5": "MD5校验值",
  "forceUpdate": "是否强制更新",
  "lastUpdated": "最后更新日期"
}
```

## 常见问题

### Q1: 无法连接到 OpenClaw 服务
**可能原因**：
- 地址格式错误（需包含协议，如 `http://` 或 `https://`）
- 网络连接问题
- 服务器未运行或防火墙拦截

**解决方案**：
- 检查地址格式是否正确
- 确认网络连接正常
- 验证服务器状态

### Q2: 登录信息保存失败
**可能原因**：
- 存储权限未授予
- 存储空间不足
- 文件系统错误

**解决方案**：
- 检查并授予存储权限
- 清理存储空间
- 重启应用重试

### Q3: WebView 显示异常
**可能原因**：
- OpenClaw 页面兼容性问题
- 缓存数据冲突

**解决方案**：
- 清理应用缓存
- 检查 OpenClaw 版本兼容性
- 联系技术支持

### Q4: 更新失败
**可能原因**：
- 网络连接不稳定
- 下载地址失效
- 存储空间不足

**解决方案**：
- 检查网络连接
- 手动访问更新地址
- 清理存储空间
